# Implementation Process and Methodology

Use this alias when the reference page is explicitly about implementation method.

1. **Discovery**
   Clarify the real problem and scope.
2. **Design**
   Pick the structure and decision model.
3. **Pilot**
   Test the smallest workable version.
4. **Scale**
   Roll out what already proved itself.
