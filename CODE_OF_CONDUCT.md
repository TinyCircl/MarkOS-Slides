# Code of Conduct

MarkOS 希望成为一个让人愿意长期协作的项目。

## 我们希望的协作方式

在这里，我们鼓励：
- 尊重不同经验水平的参与者
- 以问题为中心，而不是以人为中心
- 对实现保持严格，对表达保持克制
- 在讨论分歧时给出具体上下文、代码位置和复现方式
- 默认把别人当成真诚协作者

## 不接受的行为

这里不欢迎：
- 人身攻击
- 侮辱、羞辱、嘲讽式沟通
- 持续骚扰
- 恶意挑衅或故意破坏协作
- 泄露他人隐私信息

## 维护者的处理方式

维护者可以根据情况：
- 提醒并要求调整沟通方式
- 关闭不合适的讨论
- 拒绝或移除不适合继续参与的内容

## 报告问题

如果你在项目协作中遇到不舒服或不安全的情况，请私下联系仓库维护者，而不是在公开讨论里继续升级冲突。

这个文档的目标很简单：
- 让讨论更聚焦
- 让协作更稳定
- 让大家更愿意继续贡献
